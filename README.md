# LevelPlay Voodoo Adapter Swift Package

This Swift Package provides the LevelPlay Voodoo adapter for iOS.

## Requirements

- iOS 12.0+
- Swift 5.0+
- Xcode 15.0+

## Installation

### Swift Package Manager

Add this package to your project using Xcode:

1. File > Add Package Dependencies
2. Enter the repository URL: `https://github.com/ironsource-mobile/LevelPlay-Voodoo-Adapter-Swift-Package`
3. Select the version you want to use

Or add it to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/ironsource-mobile/LevelPlay-Voodoo-Adapter-Swift-Package", from: "5.0.0")
]
```

## Dependencies

This package includes:
- LevelPlay iOS SDK (Unity Mediation SDK)
- Voodoo SDK
